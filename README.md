# 🏏 IPL Data Analysis Dashboard (Excel + Power Pivot)
## 📌 Overview

This project presents an interactive IPL (Indian Premier League) Dashboard built using Microsoft Excel. It analyzes match and player data to uncover insights about team performance, player statistics, and match outcomes.

The dashboard leverages Power Pivot, Pivot Tables, and Data Visualization to provide a clear and interactive analytical experience.

## 🎯 Objectives
- Analyze IPL match data across seasons
- Identify top-performing teams and players
- Track trends in runs, wickets, and match results
- Build an interactive dashboard for quick insights

## 📁 Dataset

The dataset includes IPL-related information such as:

- Match details (season, venue, teams)
- Player performance (runs, wickets, strike rate)
- Match outcomes (winner, margin)

## 🛠️ Tools & Technologies

- Microsoft Excel
- Power Pivot
- Pivot Tables
- Data Cleaning & Transformation
- Data Visualization (Charts, Slicers, Dashboard)

## 📊 Key Features

- ✅ Interactive IPL Dashboard
- ✅ Team-wise performance analysis
- ✅ Player statistics (Top scorers, wicket-takers)
- ✅ Season-wise trends
- ✅ Match result insights
- ✅ Dynamic filtering using slicers

## 📈 Key Insights

- Certain teams consistently perform well across seasons
- Top players contribute significantly to match outcomes
- Batting-first vs chasing trends can be observed
- Venue plays an important role in match results

## 📌 Project Structure

- Raw Data – Original IPL dataset
- Processed Data – Cleaned and transformed data
- Pivot Tables – Analytical summaries
- Dashboard – Final interactive visualization
